# 动态规划 Dynamic Programming

- 动态规划特点
  - 重叠子问题
  - **状态转移方程**
  - 最优子结构
- 解题套路
  - 明确【状态】
  - 明确【选择】
  - 明确 dp 函数 / 数组的定义
  - 明确 base case
- 题型：*求最值*
- 核心：穷举
- 时间复杂度：递归函数调用次数 * 递归函数本身的复杂度

### 框架

- ```java
  # 初始化 base case
  dp[0][0][...] = base
  # 进行状态转移
  for 状态1 in 状态1的所有取值：
      for 状态2 in 状态2的所有取值：
          for ...
              dp[状态1][状态2][...] = 求最值(选择1，选择2...)
  ```
