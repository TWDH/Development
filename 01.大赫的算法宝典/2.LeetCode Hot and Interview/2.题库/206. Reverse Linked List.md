## 1. 翻转链表

- https://labuladong.gitee.io/algo/2/17/17/

- ```java
  class Solution {
      public ListNode reverseList(ListNode head) {
          // pre:
          ListNode pre = null;
          ListNode cur = head;
          ListNode nxt = head;
  
          while(cur != null){
              nxt = cur.next;
              cur.next = pre;
              pre = cur;
              cur = nxt;
          }
          return pre;
      }
  }
  ```
  
- ```java
  // Easy
  private void reverseLinkedList(ListNode head) {
      // 也可以使用递归反转一个链表
      ListNode pre = null;
      ListNode cur = head;
  
      while (cur != null) {
          ListNode next = cur.next;
          cur.next = pre;
          pre = cur;
          cur = next;
      }
  }
  ```
