## 1.二分法

- 注意：右边界平方时可能超出边界

- ```java
  class Solution {
      public int mySqrt(int x) {
          int left = 0;
          int right = x;
  
          while(left + 1 < right){
              int mid = left + (right - left) / 2;
              if((long) mid * mid > x){
                  right = mid;
              }
              else{
                  left = mid;
              }
          }
  
          if((long) right * right <= x){
              return right;
          }
          return left;
      }
  }
  ```
