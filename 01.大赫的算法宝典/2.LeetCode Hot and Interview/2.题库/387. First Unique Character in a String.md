- ```java
  class Solution {
      public int firstUniqChar(String s) {
          int len = s.length();
          int[] count = new int[26];
  
          for(int i = 0; i < len; i++){
              int n = s.charAt(i) - 'a';
              count[n]++;
          }
  
          for(int i = 0; i < len; i++){
              int n = s.charAt(i) - 'a';
              if(count[n] == 1){
                  return i;
              }
          }
  
          return -1;
      }
  }
  ```
