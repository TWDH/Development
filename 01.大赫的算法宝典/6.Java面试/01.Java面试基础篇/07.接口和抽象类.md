# 07.接口和抽象类

- **抽象类**可以存在**普通成员函数**，而**接口**中只能存在 **public abstract** 方法。 
- **抽象类**中的成员变量可以是各种类型的，而**接口**中的成员变量只能是 **public static final** 类型的。 
- 抽象类只能继承一个，接口可以实现多个．
