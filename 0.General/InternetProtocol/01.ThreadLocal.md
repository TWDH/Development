## 1. 使用方法

```java
// 方法
public T get() {}

```



```java
ThreadLocal<String> threadLocalA= new ThreadLocal<String>();

线程1： threadLocalA.set("1234");
线程2： threadLocalA.set("5678");
```

像‘1234’， ‘5678’这些值都会放到自己所属的线程对象中。”
