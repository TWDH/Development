# 01.简单入门

## 1. 第一个程序

- 如果没有 `@ResponseBody`, 跳转到 `hello` 页面

```java
@Controller
@RequestMapping("index")
public class indexController {
    @GetMapping
    @ResponseBody
    public String hello() {

        //如果没有@ResponseBody, 跳转到hello页面
        return "hello";
    }
}
```
