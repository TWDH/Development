## 2. 导航栏

- 链接：`<li>` + `<a>` 

- ```html
  <div>
      <ul>
          <li><a href=""></a></li>
      </ul>
  </div>
  ```

- ul li上下垂直居中

  - ```css
    .subnav ul li {
      height: 45px;
      /* 上下垂直居中 */
      line-height: 45px;
    }
    ```
