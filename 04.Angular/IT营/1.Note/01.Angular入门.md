# 01.Angular入门

## 1. 安装并创建项目

1. 安装脚手架：`npm install -g @angular/cli`
2. 创建项目：`ng new angularDemo`
   - 创建项目，不安装依赖：`ng new xxx --skip-install`
3. 安装依赖：`npm install`
4. `cd` 进入 angular 文件目录
5. 打开项目：`ng serve --open`
